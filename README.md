# dsa-problems
Project Overview

This project contains comprehensive implementations of fundamental Data Structures and Algorithms in C++.
Each program is menu-driven and allows the user to perform standard operations related to the respective data structure.

The assignment covers:

Arrays
Linked Lists
Singly Linked List
Doubly Linked List
Stacks
Queues
Trees
Sorting Algorithms

The programs were developed according to the assignment requirements provided in the documentation.
